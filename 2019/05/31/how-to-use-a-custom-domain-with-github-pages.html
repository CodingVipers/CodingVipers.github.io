How to use a custom domain with GitHub pages

31 May 2019 by CodingVipers

I bought a domain (codingvipers.com) and wanted to setup a Jekyll blog on GitHub pages.

This is what I’m trying to achieve.

  1. Custom domain should works with GitHub HTTPS encryption.
  2. Default should be apex/naked domain not www sub domain (https://codingvipers.com).
  3. www sub domain should redirected to apex domain.
  4. codingvipers.github.io should redirected to apex domain.

To get things done, just follow bellow steps.

Custom domain options in GitHub page repository

After you save those options, it’s take some time to take the effect.

CNAME file in the repository

CNAME file should contain apex custom domain without trailing newline.

DNS records

4 A name records and 1 CNAME record.

After create those DNS records, just wait at least 24 hours to take the effect (Most of the time it will only takes few hours).

Thats all you have to do.
